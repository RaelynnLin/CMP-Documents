## 四、發現問題

### 🐛 Bug #1：序號管制驗證行為與欄位定義不一致

| 項目 | 內容 |
|------|------|
| **嚴重度** | 🟡 Medium |
| **發現案例** | TC-4308-004 |
| **問題描述** | 「序號管制」欄位在 UI 上**無** `*` 必填標記，但點擊送出時仍顯示錯誤提示「請選擇序號管制」 |
| **預期行為** | 序號管制為非必填，不填寫時不應阻擋送出 |
| **實際行為** | 序號管制未填寫時顯示紅色錯誤提示，阻擋送出 |
| **影響範圍** | 所有 4 張料件卡片的序號管制欄位均受影響 |
| **建議** | 1. 若序號管制為必填 → 應加上 `*` 標記<br>2. 若序號管制為非必填 → 應移除驗證邏輯 |

### 📝 Observation #1：確認 Dialog 未列出具體料件編號

| 項目 | 內容 |
|------|------|
