## 三、測試前準備

1. **登入 UAT**：開啟 `https://cmp-uat-100.metaage.com.tw`，等待使用者輸入帳密。
2. **身份切換**：登入後於右上角切換至「業務」身份；TC18–TC19 需另切「唯讀」或無編輯權之身份。
3. **準備測試訂單**（依場景準備四張，分別處於不同階段）：
   - **訂單 A**：狀態為「採購審核 / 已核准」且匯率類型為 `CUSTOM`（自訂）。
    (https://cmp-uat-100.metaage.com.tw/main/orders/20260526653468)
   - **訂單 B**：狀態為「採購審核 / 已核准」且匯率類型為 `BKTW`（台銀）。
    (https://cmp-uat-100.metaage.com.tw/main/orders/20260526416578)
   - **訂單 C**：狀態為「訂單變更中」（`ApprovalStatus.update`），用於驗證 Save 按鈕隱藏。
    (https://cmp-uat-100.metaage.com.tw/main/orders/20260526780185)
   - **訂單 D**：已同步 T100 / 已出帳，驗證「特定狀態」下仍可直編。
    (https://cmp-uat-100.metaage.com.tw/main/orders/20260413533615)
4. **安裝 XHR 攔截器**（CLAUDE.md 第八章），URL pattern 用 `/fields`：
   ```js
   if (u.indexOf('/fields') > -1) { /* capture */ }
   ```
5. **截圖目錄**：`documents/CMP-4417-訂單開放修改訂單匯率/screenshots/`。
